# 3-Minute Demo Script

## 0:00 - 0:20

Show Obsidian with the Video Summary settings page open. Run the setup check and show that the
workflow version, webhook path, and model labels match.

## 0:20 - 0:50

Open `sample-input-url.md`. Point out the simple frontmatter: `link` and `status`.

## 0:50 - 1:40

Run `Video Summary: summarize current note`. Show the status change and explain that n8n handles
download, transcript extraction, and note generation.

## 1:40 - 2:20

Show the generated note: summary, headings, table, and action list. Emphasize that the output is
saved back into Obsidian.

## 2:20 - 2:45

Open the platform matrix. Explain that local files and pasted transcripts are the most reliable,
while short-video platforms may require cookies.

## 2:45 - 3:00

Show the install guide and GitHub release assets: `main.js`, `manifest.json`, `styles.css`, and
`Obsidian Video Summary.json`.
